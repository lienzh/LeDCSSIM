
## SAMA 风格组态画布规范（Phase 2 核心指导）

> 本节是 Claude Code 在修改组态画布 UI 时**必须遵守**的视觉与交互规范。
> 目标：让画布外观和操作体验接近真实 DCS 工程师站（如 Ovation、科远 CCMStudio），而不是通用的"网页流程图"。

### 设计原则

1. **工程制图风格，不是网页风格**：画布背景用浅灰网格（类似方格纸），不用白色空白。连线用直角折线，不用贝塞尔曲线。整体观感应像工程图纸，不像 Web App。
2. **信号从左到右流动**：输入端口一律在块的左侧，输出端口一律在块的右侧。这是 SAMA 图的核心布局约定，不可违反。
3. **紧凑排列**：功能块之间间距适中（约 40-60px），不要 Drawflow 默认的大间距散落布局。对齐到网格。
4. **信息密度高**：每个块上要能一眼看到类型名、实例名、关键参数值，不需要点开弹窗才能看到。

### 功能块节点渲染规范

```
┌─────────────────────────────┐
│  [类型图标]  PID_001         │ ← 标题栏：深色背景，白字，显示实例名
├─────────────────────────────┤
│                             │
│  IN1 ●─────────────── ● OUT │ ← 端口：左侧输入，右侧输出
│  IN2 ●                      │    端口用实心小圆点（●），直径 8px
│   SP ●                      │    端口名紧贴圆点旁边，字号 11px
│                             │
│  Kp=1.2  Ti=30s             │ ← 参数区：灰色小字显示关键参数
│                             │
│        PID                  │ ← 中心大字显示功能块类型
└─────────────────────────────┘
```

**具体尺寸要求**：
- 块最小宽度 120px，高度根据端口数自适应（每个端口占 28px 行高）
- 标题栏高度 24px，字号 12px 加粗
- 端口圆点直径 8px，圆点中心距块边缘 0px（圆点骑在边框上）
- 端口标签字号 11px，颜色 #666
- 类型名字号 14px，居中显示
- 参数值字号 10px，颜色 #999

**颜色编码（按功能块分类）**：
- 输入/输出（IO 类）：蓝色系 `#3B82F6` 标题栏
- 数学运算（SUM/MUL/DIV 等）：绿色系 `#10B981`
- 控制（PID/手自动站）：橙色系 `#F59E0B`
- 逻辑（AND/OR/NOT/比较器）：紫色系 `#8B5CF6`
- 信号处理（滤波/限幅/选择器）：青色系 `#06B6D4`
- 定时器/计数器：红色系 `#EF4444`

### 连线规范

- **线型**：直角折线（Manhattan routing），不用贝塞尔曲线。Drawflow 默认是贝塞尔曲线，必须覆盖为 `curvature: 0` 或自定义直角路径。
- **线宽**：模拟量信号 2px 实线，开关量信号 2px 虚线（dash: 5,3）
- **颜色**：默认 `#64748B`（蓝灰色），选中时 `#3B82F6`
- **箭头**：连线末端（靠近输入端口）显示小三角箭头，表示信号方向
- **交叉**：两条线交叉时，用小半圆弧跳过，表示无连接（SAMA 图传统画法）

### 画布交互规范

- **缩放**：Ctrl+滚轮缩放，范围 25%-400%，当前缩放比例显示在右下角
- **平移**：中键或空格+左键拖拽画布
- **对齐**：拖拽块时自动吸附到 20px 网格
- **多选**：框选或 Ctrl+点击多选，支持批量移动
- **连线操作**：从输出端口拖出连线到输入端口，连线过程中显示临时线
- **删除**：选中后 Delete 键删除，连线和块分别删除
- **右键菜单**：块上右键 → 编辑参数/复制/删除/查看帮助

### 功能块参数编辑

- 双击块打开参数编辑面板（**侧边滑出面板**，不是弹窗遮罩）
- 面板显示所有参数，每个参数一行：参数名 | 值 | 单位 | 范围
- 修改后实时更新块表面显示的关键参数
- ESC 或点击画布空白处关闭面板

### 仿真运行态显示

- 运行中，每个端口旁边显示当前实时值（小字，保留 2 位小数）
- 运行中，连线上可选显示信号值（鼠标悬停或常驻显示模式切换）
- 运行中，功能块不可拖拽移动（锁定画布），但可以修改参数（在线调参）
- 信号值异常（NaN/Inf/超限）时，对应端口或连线变红色闪烁

### Drawflow 定制要点

Drawflow 是轻量级库，以下是让它接近 SAMA 风格的关键改造点：

1. **自定义节点 HTML**：不用 Drawflow 默认的 `<div class="title">` 结构，完全自定义节点 HTML 模板，实现上述块渲染规范
2. **覆盖连线渲染**：设置 `curvature: 0`，或用 CSS 覆盖 SVG path 为直线段。如 Drawflow 无法原生支持直角折线，用 `createConnection` 钩子自定义 path 计算
3. **端口位置**：Drawflow 默认端口在块上下，必须改为左（输入）右（输出）。通过 CSS 将 `.input` 移到左侧、`.output` 移到右侧
4. **背景网格**：画布 `background` 用 CSS 重复背景实现 20px 方格网格
5. **节点数据结构**：每个节点的 `data` 字段存储完整的功能块定义（类型、参数、端口列表），不仅仅是 Drawflow 默认的 `name`

### JSON 数据结构（与 GraphRunner 对接）

画布导出的 JSON 必须能被 `graph_runner.py` 直接解析执行。每个节点的数据结构：

```json
{
  "id": "node_1",
  "block_type": "PID",           // 对应 block_library.yaml 中的类型名
  "instance_name": "PID_001",    // 用户自定义实例名
  "params": {                    // 功能块参数
    "Kp": 1.2,
    "Ti": 30,
    "Td": 0
  },
  "inputs": {                    // 输入端口连接关系
    "PV": {"from_node": "node_3", "from_port": "OUT"},
    "SP": {"from_node": "node_2", "from_port": "OUT"}
  },
  "position": {"x": 340, "y": 180}
}
```

### 常见错误（不要犯）

- ❌ 不要让端口在块的上方或下方（那是 Simulink 风格，不是 SAMA）
- ❌ 不要用圆角卡片式节点样式（那是 Node-Red / Web 风格）
- ❌ 不要用曲线连线（SAMA 图只有直线/直角折线）
- ❌ 不要把参数全部隐藏到弹窗里（工程师需要一眼看到关键参数）
- ❌ 运行态不要允许拖拽移动块（防止误操作）
- ❌ 不要在块内部放按钮、下拉框等 Web 控件（块表面只显示信息，编辑在侧边面板）
