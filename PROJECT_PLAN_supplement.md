
## Phase 2 详细计划：SAMA 风格组态画布升级

### 2.1 现状问题

当前 Drawflow 画布存在以下问题（需逐一解决）：

| 问题 | 现状 | 目标 |
|------|------|------|
| 节点外观 | Drawflow 默认圆角卡片风格 | SAMA 矩形功能块，左入右出端口布局 |
| 连线样式 | 贝塞尔曲线 | 直角折线（Manhattan routing） |
| 端口位置 | 上下排列或简单左右 | 严格左侧输入、右侧输出，端口名标注 |
| 信息密度 | 只显示块名 | 显示类型、实例名、关键参数值 |
| 画布背景 | 空白 | 方格纸网格 |
| 参数编辑 | 弹窗或内嵌表单 | 侧边滑出面板 |
| 运行态 | 无实时值显示 | 端口旁显示实时信号值 |
| 对齐 | 自由拖放 | 网格吸附 |

### 2.2 实施步骤

#### Step 1：功能块节点模板重写（canvas-engine.js）

**改什么**：完全替换 Drawflow 的 `addNode()` 使用的 HTML 模板

**具体要求**：
- 每个块类型生成对应的 HTML 结构：标题栏 + 端口区 + 参数区 + 类型标签
- 端口数量和名称从 `block_library.yaml` 动态读取
- 标题栏背景色按分类着色
- 端口圆点用绝对定位骑在块边框上
- 块宽度根据端口名最大长度和参数文本自适应，最小 120px

**验收标准**：拖一个 PID 块到画布上，外观符合 CLAUDE.md 中的 ASCII 示意图

#### Step 2：连线样式改造（canvas.css + canvas-engine.js）

**改什么**：覆盖 Drawflow 默认的 SVG 曲线渲染

**具体要求**：
- 方案 A（推荐）：设置 Drawflow 的 `curvature` 为 0，连线变直线，再用 CSS/JS 在拐点处插入直角段
- 方案 B（备选）：如 Drawflow 无法支持直角，用 `createConnection` 事件钩子，自行计算 Manhattan 路径替换 SVG path 的 `d` 属性
- 连线颜色、线宽、虚线区分按 CLAUDE.md 规范执行

**验收标准**：两个块横向排列，连线是水平直线；两个块有纵向偏移时，连线是 L 形或 Z 形直角折线

#### Step 3：画布基础交互完善（canvas.css + canvas-engine.js）

**改什么**：背景网格、网格吸附、缩放显示

**具体要求**：
- CSS background 实现 20px 方格网格（浅灰线 #E5E7EB，背景 #F9FAFB）
- 节点拖拽释放时 snap to grid（坐标取整到 20 的倍数）
- 右下角常驻显示当前缩放比例（如 "100%"）
- Ctrl+滚轮缩放，范围 25%-400%

**验收标准**：画布有明显的方格纸背景，块拖拽后自动对齐到网格交叉点

#### Step 4：参数编辑面板（ib.html + canvas-engine.js）

**改什么**：双击块时打开右侧滑出面板

**具体要求**：
- 面板从画布右侧滑出，宽度 320px，不覆盖全屏
- 面板内容：实例名（可编辑）、块类型（只读）、全部参数列表（参数名/值/单位/范围）
- 修改参数后，块表面的参数显示文本实时更新
- ESC 或点画布空白关闭面板
- 不用 modal 弹窗，不用浏览器 prompt()

**验收标准**：双击 PID 块 → 右侧滑出面板显示 Kp/Ti/Td 参数 → 修改 Kp 为 2.0 → 块表面显示更新为 Kp=2.0

#### Step 5：仿真运行态叠加显示（canvas-engine.js）

**改什么**：运行时在画布上叠加实时信号值

**具体要求**：
- 仿真引擎每步回传所有端口的当前值
- 每个端口圆点旁边显示当前值（字号 10px，绿色，保留 2 位小数）
- 值为 NaN/Inf 时显示红色 "ERR"
- 运行态画布锁定不可拖拽移动块
- 工具栏增加 "编辑模式/运行模式" 切换按钮

**验收标准**：点击运行 → 画布上每个端口旁出现数值 → 数值随仿真步进实时刷新 → 停止后数值消失，恢复编辑模式

### 2.3 文件修改清单

| 文件 | 修改内容 |
|------|----------|
| `static/js/canvas-engine.js` | 节点模板、连线逻辑、网格吸附、运行态显示、参数面板联动 |
| `static/css/canvas.css` | 方格背景、SAMA 块样式、端口样式、连线颜色、面板动画 |
| `templates/ib.html` | 右侧参数编辑面板 HTML 结构、编辑/运行模式切换按钮 |
| `web/app.py` | 仿真运行 API 增加端口实时值返回字段 |
| `sim_engine/graph_runner.py` | 每步执行后收集所有端口当前值供前端查询 |

### 2.4 不做的事情（Phase 2 范围外）

- 不做撤销/重做（Undo/Redo）—— 后续再加
- 不做多图纸/分页 —— 当前单画布够用
- 不做自动布局算法 —— 手动摆放
- 不做连线标注文字 —— 悬停看值就够了
- 不做块的旋转 —— SAMA 图块不旋转
